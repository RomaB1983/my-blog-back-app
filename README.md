# Spring Boot -приложение blog-back-app реализует backend для работы с постами, комментариями и файлами.

## Приложение построено по Шаблону проектирования «Controller → Service → Repository»

- JdbcTemplate-работа с БД (БД Postgress)
- Rest-API - взаимодейстивие с фронтом
- MultiPartFile - Загрузка и получение файлов

## Используемые технологии
- Java 21
- Spring Boot
- Spring Boot Test
- Servlet API
- JUnit 5
- Mockito
- Gradle

## Структура проекта
- controller — контроллеры для работы с постами и файлами
- dto - объекты для передачи запросов/ответов
- model — доменные модели
- service — бизнес-логика
- repository — работа с БД Postgres
- utils — вспомогательные классы

## Настройки
`src/main/resources/application.yaml`

## Схема БД Postgress инициализируется автоматически
`src/main/resources/schema.sql`

## Сборка JAR файла
gradle clean bootJar

## Запуск тестов
gradle test

##  Реализованные API
- Получение ленты постов (с пагинацией и поиском)
- Получение поста по ID
- Создание нового поста
- Обновление поста
- Удаление поста
- Увеличение количества лайков
- Получение комментариев к посту
- Получение конкретного комментария
- Добавление комментария к посту
- Обновление комментария
- Удаление комментария
- Загрузка изображения для поста
- Получение изображения поста
